## Editorial rules

- Keep each title under about 60 characters where possible.
- Keep each meta description under about 155–165 characters where possible.
- Use one canonical URL per page.
- Do not stuff keywords. Prefer clear institutional language.
- Keep “UCAISM Global Initiative” as the unifying brand name.
