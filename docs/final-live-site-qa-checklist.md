# Final Live Site QA Checklist

## Visual QA

- [ ] Header appears correctly.
- [ ] Footer appears correctly.
- [ ] Cards, buttons, grids, and filters display consistently.
- [ ] Text is readable on desktop and mobile.
- [ ] No horizontal scrolling on mobile.

## Functional QA

- [ ] Page-level search/filter scripts work where included.
- [ ] Links point to valid internal URLs.
- [ ] Download or placeholder links are clearly labeled.
- [ ] Contact path is discoverable.

## Technical QA

- [ ] No console errors on priority pages.
- [ ] No missing CSS or JS file errors.
- [ ] No broken image paths.
- [ ] `robots.txt` references the correct sitemap.
- [ ] `sitemap.xml` includes priority pages.

## SEO/social QA

- [ ] Title and description are appropriate.
- [ ] Canonical URL is correct.
- [ ] Open Graph metadata is ready for insertion on priority pages.
- [ ] Twitter Card metadata is ready for insertion on priority pages.
- [ ] Schema.org snippets remain factually accurate.

## Governance QA

- [ ] Research pages do not overclaim publication status.
- [ ] Media pages do not imply press coverage that does not exist.
- [ ] Events pages use careful placeholder language for future events.
- [ ] Newsletter page does not imply active mailing infrastructure until connected.
