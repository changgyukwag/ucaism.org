## Severity guide

### High

Fix before public sharing.

Examples:
- Homepage broken
- 404 on major page
- Navigation unusable
- Major mobile layout failure
- Wrong author/book/institution identity

### Medium

Fix soon, but does not block basic site launch.

Examples:
- Social preview incomplete
- Minor spacing issue
- One filter button not working
- Missing metadata on lower-priority page

### Low

Record for later maintenance.

Examples:
- Wording refinement
- Future PDF placeholder
- Minor cosmetic polish
