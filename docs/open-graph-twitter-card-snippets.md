# Phase E-3 — Open Graph & Twitter Cards

This package adds social sharing metadata foundations for UCAISM.org.

Use `docs/open-graph-twitter-card-snippets.html` as the source for page-level `<head>` integration.

## Recommended validation

After deployment, test the representative card image:

`https://ucaism.org/assets/images/social/ucaism-social-card.svg`

Then validate important pages with LinkedIn Post Inspector, Facebook Sharing Debugger, and Twitter/X Card Validator where available.

## Notes

- The default card is an SVG at 1200 × 630 ratio.
- For maximum compatibility with all external platforms, a future phase may add PNG fallback images.
- This package preserves LF line endings through `.gitattributes`.
