# UCAISM.org Phase D-9 — Events

This package adds a dedicated Events hub to the UCAISM.org GitHub Pages site.

## Install

1. Copy all files and folders from `ucaism_phase_d9_events` into the root of the existing `ucaism.org` repository.
2. Overwrite matching files when prompted.
3. Do not delete existing files.
4. Confirm that `.gitattributes` remains in the repository root.
5. Review changes in GitHub Desktop.
6. Commit and push.
