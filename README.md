# UCAISM.org Phase D-8 — Newsletter

This package adds the Phase D-8 Newsletter page to the UCAISM.org GitHub Pages site.

## Install

1. Copy all files and folders from this package into the root of the existing `ucaism.org` repository.
2. Overwrite matching files when prompted.
3. Do not delete existing site files.
4. Confirm `.gitattributes` remains at the repository root.
5. Commit and push through GitHub Desktop.

## URL after deployment

`https://ucaism.org/newsletter/`
