# UCAISM.org 2.0 Phase 2 업로드 가이드

## 1. 먼저 백업
현재 GitHub 저장소의 기존 파일(index.html, README.md, CNAME, .nojekyll)을 다운로드해 보관하세요.

## 2. ZIP 압축 해제
이 ZIP 파일을 풀면 `ucaism_org_2_0_phase2` 폴더가 나옵니다.

## 3. GitHub 업로드
1. https://github.com/changgyukwag/ucaism.org 접속
2. `Add file` → `Upload files`
3. ZIP 안의 파일과 폴더 전체를 업로드
4. Commit message: `Launch UCAISM.org 2.0 Phase 2`
5. `Commit changes` 클릭

## 4. 배포 확인
오른쪽 `Deployments` 또는 `Actions`에서 github-pages 배포 성공 확인.
이후 https://ucaism.org 접속.

## 5. 반드시 교체할 항목
- Open Wins 최종 Amazon URL
- Open Wins 표지 이미지
- 저자 사진
- 공식 연락 이메일
- Newsletter 가입 폼
