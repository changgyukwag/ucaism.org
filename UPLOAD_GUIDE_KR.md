UCAISM.org 최종 글로벌 리비전 적용 방법

1. ZIP을 압축 해제합니다.
2. 압축 해제 폴더 안의 파일과 폴더 전체를 문서 > GitHub > ucaism.org에 덮어씁니다.
3. 이 패키지에는 assets/images 폴더가 없습니다. 현재 승인된 책 표지와 사진은 변경되지 않습니다.
4. GitHub Desktop에서 변경 내용을 확인합니다.
5. Summary와 Description을 입력한 뒤 Commit to main을 누릅니다.
6. Push origin을 누릅니다.
7. 약 1~3분 뒤 ucaism.org에서 Ctrl+F5로 확인합니다.
